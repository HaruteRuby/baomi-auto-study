# 更新记录

## 1.0.2 - 2026-07-18

- 修复小节最后一个视频完播后被网站自动重播，导致学习页无法退出的问题。
- 完播状态改为锁存机制，兼容 `ended` 事件、结束画面、接近结尾及播完回到开头等情况。
- 完播后立即停止自动重播，并增加学习标签页的多次关闭重试。

## 1.0.1 - 2026-07-17

- 项目许可证由 MIT 调整为 GNU GPL v3.0（`GPL-3.0-only`）。
- 增加源文件版权及 GPL 许可声明。

## 1.0.0 - 2026-07-17

- 首个 GitHub 公开版本。
- 适配 2026 年度全国保密教育线上培训课程列表页和 `/bmVideo` 学习页。
- 支持三个课程分类依次学习。
- 自动打开待学习课程、点击播放、识别完播、返回课程列表。
- 增加跨标签页通信、课程打开锁和唯一列表页控制锁。
- 增加列表加载保护及连续完成状态复核。
- 增加暂停、解除等待和重新扫描控制面板。
- 增加 GitHub 自动更新元数据。
