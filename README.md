# 中国保密在线自动学习脚本

去年的中国保密在线自动学习脚本随着网站升级已经无法正常使用，所以我结合 2026 年度全国保密教育线上培训的新页面结构，重新制作了这个版本。

脚本需要用户先登录中国保密在线，并手动进入对应培训的“课程”界面。进入后，脚本会依次检查三个课程分类，自动打开待学习课程、切换到视频学习页、点击播放，并在识别到学习完成后关闭学习页、返回课程列表继续下一课。

## 功能

- 依次学习“保密优良传统教育”“保密知识技能教育”“保密纪律教育”。
- 自动识别待学习、学习中和已完成课程。
- 自动打开视频学习页并点击网站播放器的播放按钮。
- 按正常速度播放，不快进、不伪造学时。
- 视频完成并等待网站保存进度后，自动关闭学习页并返回课程列表。
- 防止课程列表尚未加载时误切分类。
- 防止反复打开同一个课程学习页。
- 多个课程列表页同时打开时，只允许一个页面执行自动点击。
- 提供“暂停”“解除等待”和“重新扫描”控制按钮。
- 支持 Tampermonkey、Violentmonkey 的自动更新机制。

## 安装

1. 在 Chrome、Edge 等浏览器中安装 [Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/)。
2. 打开 [`baomi-auto-study.user.js`](https://raw.githubusercontent.com/HaruteRuby/baomi-auto-study/main/baomi-auto-study.user.js)。
3. 在用户脚本管理器中确认安装。

## 使用说明

1. 登录[中国保密在线](https://www.baomi.org.cn/)。
2. 手动打开“2026年度全国保密教育线上培训”。
3. 手动点击左侧的“课程”，进入课程列表界面。
   - 脚本不会自动登录。
   - 脚本不会自动搜索培训项目或替你进入课程入口。
4. 建议只保留一个课程列表页，并确认页面右侧出现“保密网自动学习”面板。
5. 脚本会按以下顺序逐类处理：
   1. 保密优良传统教育
   2. 保密知识技能教育
   3. 保密纪律教育
6. 脚本打开学习页后，会自动点击播放按钮并等待视频正常播放结束。
7. 网站保存进度后，学习页会自动关闭，原课程列表页会刷新并继续下一课。

## 控制面板

- **暂停/开始**：临时停止或恢复自动操作。
- **解除等待**：学习页被手动关闭、浏览器阻止弹窗或页面异常时，解除当前课程的等待锁。
- **重新扫描**：从第一个分类开始重新检查课程状态，不会清除网站已经记录的学习进度。

## 自动更新

脚本通过 GitHub Raw 地址提供更新。Tampermonkey 或 Violentmonkey 会按照扩展自身的更新周期检查新版本，也可以在扩展菜单中手动选择“检查用户脚本更新”。

每次发布都会同步提升脚本头部的 `@version`。`main` 分支只保存经过验证的稳定版本。

## 常见问题

### 页面没有出现控制面板

- 确认用户脚本已经启用。
- 确认当前页面是课程列表页或 `/bmVideo` 学习页。
- 刷新页面后，在开发者工具控制台查找：`[保密网自动学习] v1.0.2 已注入`。
- Chrome 新版本可能要求在扩展管理页面允许用户脚本运行。

### 反复打开学习页

关闭此前遗留的重复学习页，只保留一个课程列表页，然后刷新课程列表页。新版本会在点击课程前加锁，收到学习完成通知前不会重复点击。

### 一直显示“等待学习页完播”

先检查学习页是否仍在播放。若学习页已关闭或加载失败，点击“解除等待”，脚本会重新读取课程状态。

### 视频结束后没有返回

自动关闭学习页依赖浏览器允许网站打开和关闭课程标签页。请检查浏览器是否拦截了弹出式窗口，并在必要时手动关闭学习页。

## 使用说明与免责声明

- 本脚本只辅助页面操作，按正常播放时长学习。
- 本脚本不篡改网站接口、不伪造学时、不提供考试答案。
- 网站学习状态可能存在延迟，脚本会刷新课程列表并重新核对。
- 网站页面升级后脚本可能失效，请通过 [GitHub Issues](https://github.com/HaruteRuby/baomi-auto-study/issues) 反馈。
- 使用者应遵守所在单位的培训要求、网站规则和信息安全规定。
